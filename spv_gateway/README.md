> [!CAUTION]  
> This pup does not have a stable release yet.

This pup will install [Libdogecoin SPV Enclave Gateway](https://github.com/dogecoinfoundation/libdogecoin) as a pup on your node.

It will generate a new wallet and start block sync from the last checkpoint.

The gateway provides an interface for merchants to connect SPV to their existing site.
